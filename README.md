# School PPT Downloader

用于课程平台页面里抓取 PPT 链接并下载。脚本默认全站运行：`*://*/*`。

## 功能

- 手动扫描页面/资源，提取 `.ppt/.pptx/.pdf` 链接
- 页面右下角提供“下载PPT”按钮和列表
- 一键复制全部链接

## 安装

1. 浏览器安装 Tampermonkey 扩展
2. 新建脚本，把 `school-ppt-downloader.user.js` 内容粘进去
3. 保存后访问课程平台页面即可

## 使用

1. 打开课程平台页面
2. 右下角出现一个圆形按钮
3. 列表里点击“下载”，或点“复制”把链接扔到下载器里

默认是轻量模式：只在你点“扫描”时抓取。

如果页面是异步加载的，可以点“扫描”重新抓取一次。

需要更彻底的抓取时，用油猴菜单“切换深度抓取”，它会拦截接口响应并监听 DOM 变化。开启后建议刷新页面。

## 站点开关

默认全站运行。如果你只想在某个站点生效：

1. 打开目标站点页面
2. 点击油猴菜单里的“仅在此站点启用”

你也可以在菜单里点“查看当前模式”、“查看白名单”或“清空白名单”。

需要恢复全站时，点“启用所有网站”。

## 可能的情况

- 深度抓取会尝试从接口响应里提取
- 某些链接被限制下载时，可以复制链接后用浏览器或下载器尝试

## 注意事项

- 全站运行会在所有网站注入脚本，可能误抓到无关链接
- 如果你只想在特定站点生效，建议把 `@match` 改成目标域名
- 无痕模式是否生效取决于浏览器是否允许 Tampermonkey 在无痕中运行
