# School PPT/PDF Downloader

一个轻量的油猴脚本：帮你从课程平台页面里找出 PPT/PDF 的真实链接，并一键下载或复制。

## 为什么写这个

方便把课件整理下来，用于 AI 总结和辅助学习。有些同学找不到下载入口，所以做了这个小工具帮一把。

## 功能

- 手动扫描页面/资源，提取 `.ppt/.pptx/.pdf/.zip/.doc/.docx/.xls/.xlsx/.rar/.7z`
- 右下角小圆形按钮，点开即用
- 支持在线预览地址解码（`onlinePreview?url=...`）
- 一键复制全部链接

## 安装

1. 浏览器安装 Tampermonkey 扩展
2. 新建脚本，把 `school-ppt-downloader.user.js` 内容粘进去
3. 保存后打开目标网站即可

## 使用

1. 打开课程平台页面
2. 右下角出现一个小圆形按钮
3. 点开面板，点击“扫描”
4. 列表里点击“下载”，或用“复制”把链接丢进下载器

如果页面是异步加载的，点一次“扫描”就行。深度抓取会更慢，但更彻底。

快捷键默认是 `alt+u`，用于显示/隐藏所有悬浮窗（按钮和面板）。

面板右上角有“设置”按钮，里面可以改快捷键、切换深度抓取和站点开关。

## 深度抓取

默认是轻量模式，只有在你点“扫描”时才抓取。需要更彻底时：

- 油猴菜单 → “切换深度抓取”
- 开启后建议刷新页面再扫描

## 站点开关

默认全站运行。你可以在油猴菜单里控制范围：

- “仅在此站点启用”：只对白名单域名生效
- “查看白名单”/“清空白名单”
- “启用所有网站”：恢复全站

## 注意事项

- 全站运行会在所有网站注入脚本，可能误抓无关链接
- 无痕模式是否生效取决于浏览器是否允许 Tampermonkey 在无痕中运行
- 用完建议关闭脚本或切回“仅在此站点启用”，减少内存占用
